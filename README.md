# ゲーム内容
ザコ敵を捕まえて投げて大きなボスを倒します。
ザコ敵を捕まえるのは痛いので、掴んでいるときは体力が減っていきます。

# 操作
- 移動：WASDかカーソルキー
- 捕まえる：「C」キー（敵が前にいる時に押すと捕まえられる。離すと投げる）
